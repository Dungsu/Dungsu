## 🚀 Key Projects

### 🖥️ UPM (Universal PC Master) Decode
*PC 상태 수집 및 3-Tier 기반 IoT 모니터링 시스템*
- **기간:** 2026.03 ~ 진행 중
- **역할:** 리드 개발자 (아키텍처 설계 및 구현)
- **Tech:** `C#`, `.NET 8`, `FastAPI`, `MQTT`, `Arduino`
- **Description:** - WMI를 활용한 PC 하드웨어 성능 데이터(CPU, RAM 등) 수집
  - REST API(FastAPI) 및 MQTT 프로토콜을 이용한 데이터 통신 파이프라인 구축
  - 모바일 앱 및 Arduino 하드웨어와의 연동을 통한 실시간 모니터링 환경 제공
